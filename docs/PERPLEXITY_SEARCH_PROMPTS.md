# Perplexity Search Prompts - Nordic Circular Buildings

**Purpose:** Optimized prompts for Perplexity AI to find missing verification data

---

## Prompt Template Structure

```
I'm researching [PROJECT NAME] in [CITY, COUNTRY] for a circular buildings database.

**What I already know:**
- [Key fact 1]
- [Key fact 2]
- [Key fact 3]

**What I need to find:**
1. [Specific data point 1]
2. [Specific data point 2]
3. [Specific data point 3]

**Search in:** [Language] sources, official/government sites preferred
**Document types:** PDF reports, project documentation, LCA studies, press releases

Please provide specific numbers, percentages, and cite your sources with URLs.
```
